# Time Series Forecasting Dashboard

## Project Overview
This project is an interactive Streamlit dashboard for time series analysis and forecasting. 
Multiple forecasting models are implemented to analyze and predict future trends in the dataset.

## Models Implemented
- SARIMA (Seasonal AutoRegressive Integrated Moving Average)
- Holt-Winters Exponential Smoothing
- GARCH (Generalized Autoregressive Conditional Heteroskedasticity)
- Machine Learning based forecasting

## Features
- Interactive data visualization
- Forecast comparison across models
- Trend and seasonality analysis
- Model prediction visualization

## Tools Used
Python
Streamlit
Pandas
Statsmodels
Scikit-learn
Plotly

## Running the Application
pip install -r requirements.txt
streamlit run app.py
